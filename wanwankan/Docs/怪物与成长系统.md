## 🎁 掉落系统

### 掉落率计算

```
基础掉落率 = 物品设定概率
实际掉落率 = 基础掉落率 × (1 + 幸运值 × 0.02)

例：幸运值10，30%基础率
实际 = 30% × 1.2 = 36%
```

### 金币掉落公式

```
金币 = Random(最小值, 最大值) × (1 + 贪婪之眼效果)

贪婪之眼：金币×2
```

### 经验掉落公式

```
实际经验 = 基础经验 × 等级修正

等级修正：
- 怪物等级 > 玩家等级+5：×1.2
- 怪物等级 = 玩家等级±5：×1.0
- 怪物等级 < 玩家等级-5：×0.5
- 怪物等级 < 玩家等级-10：×0.1
```
