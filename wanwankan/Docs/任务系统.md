## 🎯 系统概述

### 核心设计理念

**一周目可完成所有非互斥任务，二周目可完成所有互斥任务的另一条线。**

### 任务分类

| 类型 | 说明 | 标记 |
|------|------|------|
| **互斥任务** | 两个独立任务，完成A会永久锁死B | 🔒 |
| **分支任务** | 同一任务内的不同选择，选一个即可 | 🔀 |
| **连锁任务** | 需要前置任务/物品，但不互斥 | 🔗 |
| **普通任务** | 无限制，可自由完成 | ✅ |

### 周目完成统计

| 统计项 | 数量 |
|--------|------|
| 任务总数 | 59个 |
| 互斥任务对 | 8对（16个任务） |
| 非互斥任务 | 43个 |
| **一周目可完成** | **51个**（43非互斥 + 8互斥任务各选一边） |
| **二周目新增** | **8个**（互斥任务的另一边） |

## ⚙️ 核心机制

### 1. 互斥任务系统 🔒

完成任务A会**永久锁死**任务B，需要二周目才能体验另一条线。

```
┌─────────────────┐          ┌─────────────────┐
│  S01: 森林猎手   │ ← 互斥 → │  S02: 森林守护者 │
│  奖励: 猎人之弓  │          │  奖励: 自然祝福  │
│  善恶: -35      │          │  善恶: +35       │
└─────────────────┘          └─────────────────┘
      一周目选这个     OR      二周目选这个
```

### 2. 分支选择系统 🔀

同一任务内的不同选择，**不影响其他任务**，一周目就能体验（只是只能选一个结局）。

```
S08: 酒馆的秘密
├── 第1-3环: 调查走私据点
└── 第4环: 做出选择
    ├── A: 举报（善恶+15）
    └── B: 接管（善恶-20）
    
→ 无论选A或B，其他任务不受影响
```

### 3. 连锁任务系统 🔗

需要完成前置任务或获取特定物品，但**不与其他任务互斥**。

```
S03: 失踪的孩子（隐藏环获得【人贩子徽章】）
          ↓ 连锁
S41: 人贩子的踪迹（需要【人贩子徽章】）

→ 只要完成S03隐藏环，S41就可以做
→ 不需要放弃任何其他任务
```

### 4. 善恶度系统

| 善恶值 | 范围 | 称号 | 效果 |
|--------|------|------|------|
| 至善 | +80 ~ +100 | 圣者 | 神殿折扣50%、神圣物品解锁 |
| 善良 | +40 ~ +79 | 善人 | NPC好感提升 |
| 中立 | -39 ~ +39 | 旅人 | 无特殊效果 |
| 邪恶 | -40 ~ -79 | 恶徒 | 黑市折扣50% |
| 至恶 | -80 ~ -100 | 堕落者 | 禁忌技能解锁 |
