## ⚙️ 核心机制

### 1. 好感度系统

每个NPC都有隐藏的好感度值，影响对话选项和奖励。

| 好感度 | 范围 | 效果 |
|--------|------|------|
| 敌对 | -100 ~ -50 | 拒绝交易、可能攻击 |
| 冷淡 | -49 ~ -1 | 对话简短、价格上涨 |
| 中立 | 0 | 正常交互 |
| 友好 | 1 ~ 49 | 额外对话、折扣5% |
| 亲密 | 50 ~ 99 | 特殊任务、折扣15% |
| 挚友 | 100 | 专属奖励、折扣30% |

**好感度获取方式**：
- 完成NPC相关任务：+10~30
- 赠送喜好物品：+5~15
- 帮助其相关联的NPC：+5
- 选择善意对话选项：+1~3
- 选择恶意对话选项：-5~10
- 伤害/偷窃：-50~100

### 2. 阵营系统

NPC归属不同阵营，阵营关系影响互动。

| 阵营 | 图标 | 代表NPC | 对立阵营 |
|------|------|---------|----------|
| 冒险者公会 | ⚔️ | 公会长赫克托、各地公会员 | - |
| 命运神殿 | ✨ | 祭司艾琳娜、各地神官 | 黑骰教团 |
| 黑骰教团 | 🎲 | 神秘信徒 | 命运神殿 |
| 矮人部落 | ⛏️ | 族长铁锤、矿工们 | 宝石商人 |
| 精灵一族 | 🌿 | 长老翠叶 | 腐化势力 |
| 商人联盟 | 💰 | 马可、各地商人 | - |
| 中立 | ⚪ | 村民、流浪者 | - |

**阵营好感传递**：
- 帮助某阵营NPC，同阵营其他NPC好感+5
- 伤害某阵营NPC，同阵营其他NPC好感-20
- 阵营声望达到一定值可解锁专属商品/任务

### 3. 对话系统

**对话类型**：

| 类型 | 说明 | 例子 |
|------|------|------|
| 普通对话 | 日常问候、世界观信息 | "欢迎来到晨曦镇" |
| 任务对话 | 任务相关的对话树 | "我需要你帮我找..." |
| 商店对话 | 进入交易界面 | "看看我的货物吧" |
| 条件对话 | 需要特定条件触发 | 好感度>50时解锁 |
| 隐藏对话 | 需要特定物品/进度 | 持有某物品时触发 |

**对话选项类型**：

| 选项类型 | 标记 | 效果 |
|----------|------|------|
| 善意选项 | 💚 | 好感+、善恶值+ |
| 恶意选项 | 💔 | 好感-、善恶值- |
| 中立选项 | ⚪ | 无额外效果 |
| 检定选项 | 🎲 | 需要骰子检定 |
| 物品选项 | 📦 | 需要特定物品 |
| 金币选项 | 💰 | 需要支付金币 |
