## 🎯 系统概述

### 核心设计理念

**Roguelite + 多周目继承**：每局游戏独立，但关键物品和成就跨周目保留。

```
┌─────────────────────────────────────────────────────────┐
│  单局游戏（Roguelike层）                                 │
│  ├─ 装备、金币、技能 → 死亡后重置                        │
│  ├─ 善恶度 → 死亡后重置                                  │
│  └─ 地图进度 → 死亡后重置                                │
│                                                          │
│  跨周目继承（Meta层）                                    │
│  ├─ 灵魂碎片、命运碎片 → 永久保留                        │
│  ├─ 诅咒之种 → 永久解锁                                  │
│  ├─ 成就/称号 → 永久保留                                 │
│  ├─ 记忆残片 → 永久保留                                  │
│  └─ 结局进度 → 永久记录                                  │
└─────────────────────────────────────────────────────────┘
```

### 任务分类

| 类型 | 说明 | 标记 |
|------|------|------|
| **互斥任务** | 两个独立任务，完成A会永久锁死B | 🔒 |
| **分支任务** | 同一任务内的不同选择，选一个即可 | 🔀 |
| **连锁任务** | 需要前置任务/物品，但不互斥 | 🔗 |
| **普通任务** | 无限制，可自由完成 | ✅ |
| **继承任务** | 需要跨周目物品才能触发 | 🌀 |

## ⚙️ 核心机制

### 1. 互斥任务系统 🔒

完成任务A会**永久锁死**任务B，需要二周目才能体验另一条线。

```
┌─────────────────┐          ┌─────────────────┐
│  S01: 森林猎手   │ ← 互斥 → │  S02: 森林守护者 │
│  奖励: 猎人之弓  │          │  奖励: 自然祝福  │
│  善恶: -35      │          │  善恶: +35       │
│  诅咒: 血腥畸芽  │          │  祝福: 自然之力  │
└─────────────────┘          └─────────────────┘
      一周目选这个     OR      二周目选这个
          ↓                        ↓
    获得诅咒之种              获得祝福之种
    （可下周目激活）           （可下周目激活）
```

### 2. 分支选择系统 🔀

同一任务内的不同选择，**不影响其他任务**，一周目就能体验（只是只能选一个结局）。

```
S08: 酒馆的秘密
├── 第1-3环: 调查走私据点
└── 第4环: 做出选择
    ├── A: 举报（善恶+15）
    └── B: 接管（善恶-20）
    
→ 无论选A或B，其他任务不受影响
```

### 3. 连锁任务系统 🔗

需要完成前置任务或获取特定物品。**继承物品跨周目保留**。

```
S03: 失踪的孩子（隐藏环获得【人贩子徽章】）
          ↓ 继承物品（跨周目保留）
S41: 人贩子的踪迹（需要【人贩子徽章】）

→ 即使本局没完成S03，只要之前周目获得过徽章，S41就可以做
→ 继承物品在传承宝库中查看
```

### 4. 善恶度系统

| 善恶值 | 范围 | 称号 | 效果 |
|--------|------|------|------|
| 至善 | +80 ~ +100 | 圣者 | 神殿折扣50%、神圣物品解锁 |
| 善良 | +40 ~ +79 | 善人 | NPC好感提升 |
| 中立 | -39 ~ +39 | 旅人 | 无特殊效果 |
| 邪恶 | -40 ~ -79 | 恶徒 | 黑市折扣50% |
| 至恶 | -80 ~ -100 | 堕落者 | 禁忌技能解锁 |

> ⚠️ 善恶度**单局内累计**，死亡后重置。但达成特定善恶值通关会解锁永久成就。

### 继承机制详解

#### 灵魂碎片

- 每个章节BOSS掉落1个灵魂碎片
- 收集7个可解锁**真结局**路线
- 即使本局死亡，已获得的碎片**永久保留**

#### 命运碎片

- 完成收集者委托任务获得
- 收集5个可在**命运祭坛**制作【命运骰】
- 【命运骰】效果：每局可重投1次骰子

#### 记忆残片

- 隐藏在特殊房间、事件、对话中
- 收集完整后解锁**隐藏剧情线**
- 揭示世界观真相和埃隆的过去

### 🩸 吞噬机制详解

激活**血腥畸芽**后，解锁独特的【吞噬】技能：

#### 触发条件
- 目标怪物血量 ≤ **5%** 时可使用
- 使用后直接击杀目标并吞噬

#### 吞噬效果
```
吞噬怪物后，获得目标的【最大HP】作为奖励：

┌─────────────────────────────────────────┐
│  目标最大HP: 100                         │
│  ├─ 50% → 当前HP回复: +50               │
│  └─ 50% → 永久血量上限: +50             │
└─────────────────────────────────────────┘

示例：
  吞噬【哥布林】(HP上限20) → 回复10HP + 血上限+10
  吞噬【骨龙】(HP上限200) → 回复100HP + 血上限+100
```

#### BOSS吞噬规则

| BOSS | 可吞噬 | 说明 |
|------|--------|------|
| 骷髅王·阿萨斯 | ❌ | 灵魂被封印，无法吞噬 |
| 黄金龙·格里德 | ❌ | 身体是黄金，无血肉可吞噬 |
| 毒蔷薇·艾薇拉 | ❌ | 植物本体，无法吞噬 |
| 疯博士·维克多 | ✅ **唯一可吞噬BOSS** | 人类身体，吞噬后获得【疯狂之血】 |
| 不朽将军·铁血 | ❌ | 不死之身，无法消化 |
| 虚空使者·尼希尔 | ❌ | 虚空存在，无实体 |
| 最终BOSS·埃隆 | ❌ | 神级存在，无法吞噬 |

#### 特殊奖励：疯狂之血

吞噬**疯博士·维克多**后获得隐藏奖励：
- 【疯狂之血】：永久HP上限+150，但随机获得一个额外诅咒之种

#### 吞噬获取方式

| 获取方式 | 说明 |
|----------|------|
| 🩸 血腥畸芽 | 激活诅咒之种后永久解锁 |
| 🃏 吞噬卡牌 | 战斗奖励中小概率出现（5%） |
| 📜 吞噬技能书 | 商店/宝箱稀有掉落 |

> ⚠️ 注意：未激活血腥畸芽时，吞噬技能**可以正常使用**，但不会禁用其他回血方式。血腥畸芽的代价是**只能靠吞噬回血**。

### 嵌合畸变（特殊变异模式）

激活**嵌合畸变**后，角色攻击模式完全改变：

```
【普通模式】
├─ 普通攻击：1d20命中，1d6+STR伤害
├─ 技能：消耗气力使用
├─ 防御：减少50%伤害
└─ 骰子判定：正常规则

【嵌合模式】（激活诅咒之种后）
├─ 攻击变为3连击：每击1d4伤害，无视防御
├─ 无法使用传统技能，改为「突变技能」
│   ├─ 【撕裂】：造成出血，3回合每回合损失HP
│   ├─ 【吞噬】：目标≤5%血量时可用，吞噬获得血量（同血腥畸芽）
│   └─ 【蜕变】：消耗50%当前HP，攻击力翻倍持续3回合
├─ 无法防御，改为「适应」
│   └─ 受到伤害后，同类型伤害-20%（可叠加3次）
├─ HP低于30%时进入「狂暴状态」
│   └─ 攻击速度×2，但无法控制攻击目标
└─ 无法使用普通药剂，只能通过吞噬回血
```
